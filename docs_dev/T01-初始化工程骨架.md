# T01 初始化工程骨架

## 状态

- 已完成（2026-03-29）

## 目标

创建一个可持续演进的 `Rust + axum` 项目骨架，为后续模块开发提供统一入口。

## 输入

- `docs_design/06-service-architecture-axum.md`
- `docs_dev/README.md`

## 前置依赖

- 无

## 具体工作

1. 初始化 Cargo 项目。
2. 引入基础依赖：
   - `axum`
   - `tokio`
   - `serde`
   - `serde_json`
   - `thiserror`
   - `tracing`
   - `tracing-subscriber`
   - `sea-orm`
   - `sea-orm-migration`
3. 建立基础目录结构：
   - `src/app`
   - `src/config`
   - `src/error`
   - `src/modules`
   - `src/storage`
   - `src/web`
4. 提供最小启动入口：
   - 配置加载
   - tracing 初始化
   - router 初始化
   - http server 启动
5. 增加健康检查接口：
   - `GET /healthz`

## 建议产物

- `Cargo.toml`
- `src/main.rs`
- `src/app/mod.rs`
- `src/web/router.rs`
- `src/web/handlers/health.rs`

## 验收标准

1. `cargo run` 可以正常启动服务。
2. `GET /healthz` 返回 200。
3. 项目目录结构已具备后续分模块扩展能力。

## 不在本任务内

- 数据库连接
- 业务模型
- 版本管理流程
